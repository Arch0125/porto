# `experimental_createAccount`

Creates a new Account.

<TryItOut
  exampleSlug="#example"
  fn={async ({ provider }) => {
    const account = await provider.request({
      method: 'experimental_createAccount',
      params: [{ label: 'My Example Account' }],
    })

    return account
  }}
  requireConnection={false}
  transformResultCode={(code) => {
    return 'const account = ' + code
  }}
/>

## Request

```ts
type Request = {
  method: 'experimental_createAccount',
  params: [{ 
    /** Chain ID to create the account on. */
    chainId?: Hex.Hex
    /** Label for the account. Used as the Passkey credential display name. */
    label?: string 
  }]
}
```

## Response

Address of the created account, and its capabilities.

```ts
type Response = {
  // Address of the created account.
  address: `0x${string}`,
  // Account capabilities.
  capabilities?: {
    // Admins of the account.
    admins?: AdminsCapabilityResponse,
    // Permissions the account has authorized.
    permissions?: PermissionsCapabilityResponse
  }
}
```
